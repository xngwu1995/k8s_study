准备管理用 EC2 实例
==================

## 知识点

* 准备管理用 EC2 实例

## 实战演习/说明讲解

>看图说话

+ 编写生成 EKS 集群管理用的 EC2 CloudFormation 模版
  - 安装 AWS CLI 2.x
  - 安装 kubectl CLI 1.21
  - 安装 eksctl CLI 
+ 生成 EC2 实例, 动作确认

## 操作步骤

### 编写生成 EKS 集群管理用的 EC2 CloudFormation 模版

+ 堆栈名称: awsEKSEC2
+ 文件名称: awsEKSEC2.yml

### 生成 EC2 实例, 动作确认

```bash
# AWS CLI
$ aws --version
# kubectl K8S管理工具
$ kubectl version --short --client
# eksctl 集群管理工具
$ eksctl version
```
